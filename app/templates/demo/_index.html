With Options

A little more complete usage, with some options given via hardcoded values.

<<%= ngDirectiveName %> option1="value1" option2="value2"> > 
<<%= ngDirectiveName %>  
   option1="value1"
   option2="value2">
</<%= ngDirectiveName %>>

The following options are available:

  • option1 : description1
  • option2 : description2
  • ... : ...

Another usage, with some options given via scope values.

<div  ng-controller="MyCtrl">
   <<%= ngDirectiveName %>

    option2="option2"    
   </<%= ngDirectiveName %>>
</div>
